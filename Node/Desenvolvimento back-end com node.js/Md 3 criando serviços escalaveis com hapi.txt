------------------------------
o que é hapi?
um framework para construir aplicações e serviços
similar ao express.js
software de codigo aberto

permite que os desenvolvedores se concentrem em escrever logicca ao inves de pensar na arquitetura

------------------------------
funcionalidades

Autenticação e autorização 
armazenamento cache
roteamento
validação de schema
cookies
loggin
tratamento de erros
monitoramento de porcessos(Good lib)
------------------------------
Hapi Vs Express

Express:
Um pouco menos opinativo que o hapi, sendo mesnos abstraido do node
parece uma aplicação nativa
Usa middlewares para fornecer acesso ao pipeline de solicitação resposta

Hapi:
estrutura padrão 
implementação mais abstrata do node 
usa plugins para extender as funcionalidades
